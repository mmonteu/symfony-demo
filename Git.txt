Télécharger Git : 
	- https://git-scm.com/download/mac


Initialiser Git :
	- Se rendre dans le dossier souhaiter et taper ($ git init)
	- Identifier votre nom : ($ git config --global user.name 'Marc Monteu')
	- Identifier votre mail : ($ git config --global user.email 'marc.monteu@gmail.com')
	- Username : mmonteu

Commandes :
	- ($ git status) permet de voir les différents fichiers
	- ($ git add .) permet de sélectionner tous le dossier présent
	- ($ git commit) permet d'envoyer les fichiers la première fois, ajouter un commentaire avec la touche "i" avec l'insert
	- ($ git commit -m 'le commentaire') permet d'envoyer les changements avec un commentaire

	- ($ git push) permet d'afficher les changements sur GitHub
	- ($ git pull) permet de récupérer les changements
	- ($ git clone https://xxx.git) permet de récupérer un projet

	- ($ git branch xxx) permet de créer une branch
	- ($ git checkout xxx) permet de se diriger dans la branch en question
	- ($ git branch --list) permet de lister les différentes branch disponible
	- ($ git merge xxx) permet d'inclure les modifications de la branch en question dans la branch master


GitHub :
	- Créer un repository 
	- Ajouter GitHub au projet : 
		- ($ git remote add origin https://github.com/xxx/xxx.git)
		- ($ git push -u origin master)
	- Ajouter le fichier readme.md, il permet de visualiser du texte directement sur le projet GitHub
